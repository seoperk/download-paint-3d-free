Current Status in 2026

Microsoft officially deprecated Paint 3D and removed it from the Microsoft Store starting November 4, 2024. Existing installations continue to function normally, but new downloads are no longer available directly through the official Store app search. This decision stemmed from low usage compared to the refreshed classic Paint app and a shift toward AI-enhanced tools. However, the app was never truly "shut down"—it's still usable, and legitimate archived or third-party trusted sources provide safe access to the installer packages.

In 2026, community interest remains strong, especially among users who prefer its unique 3D doodle and sticker features over newer alternatives. Don't worry—safe download options exist if you know where to look.

Official & Safe Sources

The safest approach is using archived Microsoft Store packages from reputable generators or trusted mirror sites that host original .appx/.msix files. Avoid random third-party sites claiming "Paint 3D.exe" as they often bundle unwanted software.

  • Microsoft Store Archive Tools: Use community tools like the Microsoft Store Generation Project to generate direct download links from the official app ID (9NBLGGH5FV99). Paste the Store URL into these generators for clean, virus-free packages.
  • Trusted Mirrors: Sites like thepaint3d.com or softonic.com offer verified downloads of the latest available version (around 6.2410.x). These mirrors frequently update and scan files.
  • Softonic / Uptodown: Long-standing download portals with security checks and user reviews provide safe Paint 3D installers.

Always scan downloaded files with Windows Defender or a trusted antivirus before installing.

How to Download & Install Paint 3D

Follow these steps for a smooth installation:

  1. Visit a trusted source like Paint 3D or use a Store link generator with the app ID.
  2. Download the .appx or .msixbundle file (usually 50–100 MB).
  3. Double-click the file. Windows may prompt for App Installer—if missing, install it from the Microsoft Store first.
  4. Click "Install" and wait for completion.
  5. Launch Paint 3D from the Start menu. If issues arise, right-click the file → "Install for all users" or use PowerShell: Add-AppxPackage -Path "path\to\file.appx".

Tip: If you're on a fresh Windows install, enable sideloading in Settings → Privacy & security → For developers.

Key Features of Paint 3D

Despite its age, Paint 3D shines with:

  • Easy 2D painting with brushes, colors, and effects
  • 3D shape library (cubes, spheres, doodles that become 3D)
  • Stickers and textures that wrap around models
  • Magic Select and Crop tools
  • Export to PNG, JPEG, or 3D formats (FBX, 3MF)
  • Simple sharing to Remix3D (legacy) or social platforms

It's perfect for beginners wanting to experiment with 3D without complex software like Blender.

Alternatives if Needed

If Paint 3D doesn't suit you:

  • New Microsoft Paint: Updated with layers, AI tools, and better 2D features (free in Windows 11).
  • 3D Viewer: For viewing 3D models (still in Store).
  • Blender: Free, powerful open-source 3D suite.
  • Tinkercad: Browser-based, beginner 3D modeling.
